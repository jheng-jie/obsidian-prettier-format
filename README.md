## Obsidian Prettier Format Plugin

<br>

#### Build

- `cmd`

  ```shell
  yarn run build
  ```

- output: `./dist`

<br>

#### Obsidian Add Plugin

- copy `styles.css` `manifest.json` `main.js` to vault  `/.obsidian/plugins`

  ```shell
  cp -r dist/ ${ObsidianProject}/.obsidian/plugins/prettier-format/
  ```
